# [DP.SC.114] CRM и работа с участниками

## Обещание

**Кому:** Бухгалтер / CRM-оператор, Маркетолог.

**Зачем:** Понимание unit economics. Управление на основе данных. Проактивная работа с оттоком.

**Что получит:**
- **Воронка** — регистрация → T1 → T2 → T3 → T4, конверсия на каждом этапе
- **Удержание** — streak, last_active, engagement score; сегментация: активные / dormant / at-risk
- **Работа с оттоком** — nudge при 14+ дней без активности, exit survey при отмене
- **Контакты** — единая карточка: тир, подписки, баллы, активность, потоки, платежи
- **Отчётность** — MRR, churn rate, cohort analysis, LTV по тирам и программам

**Критерий приёмки:** Воронка визуализирована. At-risk участники выявляются автоматически. CRM-оператор видит полную карточку.
