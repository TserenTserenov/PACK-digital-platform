# [DP.SC.005] Публикация контента

## Обещание

**Кому:** Пользователь IWE (L4) — как автор; Сообщество (L1) — как читатели

**Зачем:** Публикация контента без ручной работы с Discourse. Автор готовит текст → система планирует, публикует, отслеживает реакции.

**Что получит:**
- **Автоматическое сканирование** готовых к публикации текстов (status: ready)
- **Публикация по расписанию** в Discourse (клуб) с форматированием
- **Ручная публикация** по команде /club publish
- **Уведомления о комментариях** в TG

**Критерий приёмки:** Текст со status: ready публикуется в течение запланированного слота. Автор получает уведомление о новом комментарии < 15 мин.
